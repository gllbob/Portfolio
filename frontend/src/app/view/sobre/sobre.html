Sobre Mim

Engineer & Project Leader

Minha jornada no desenvolvimento começou há seis anos com uma premissa simples: código é apenas o meio; a solução é o fim. Mais do que escrever linhas de comando, especializei-me em arquitetar ecossistemas digitais que resolvem problemas reais.

Construí minha base técnica atuando em frentes diversificadas: do desenvolvimento de sistemas com PHP e MySQL na Mitikas, até a arquitetura de aplicações globais com React e AWS na Top Serviços.

Expandi meu escopo para a Liderança Técnica na Bárbara Monteiro Arquitetura. Liderei o desenvolvimento de uma solução móvel nativa utilizando Java e APIs RESTful, entrega que resultou em uma redução de 30% no tempo de resposta ao cliente.

Hoje, concilio essa vivência de mercado com o rigor acadêmico do Bacharelado em Ciência e Tecnologia na UFBA. Busco desafios que unam engenharia de software avançada, volumes de dados e gestão técnica.
